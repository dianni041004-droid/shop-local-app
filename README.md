# Shop Local App

Aplicación móvil desarrollada con Ionic + Angular + Capacitor.

##  Descripción
Directorio de negocios locales en República Dominicana.

##  Funcionalidades
- Búsqueda de negocios
- Geolocalización
- Reseñas
- Galería de imágenes
- Contacto directo

##  Tecnologías
- Ionic Framework
- Angular
- Capacitor
- API REST
